# ICharge Website

A modern, responsive web application built with **React**, **TypeScript**, **Tailwind CSS**, and **ShadCN UI**.

---

## 🛠️ Tech Stack

- **UI Framework/Library**: [React](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **Internationalization**: i18next / React-i18next
- **Icons** : [Lucide](https://lucide.dev/)

---

## 📂 Folder Structure

```text
.
├── src/
│   ├── assets/         # Static media (images, icons, svgs, global styles)
│   ├── components/     # Reusable UI components (buttons, inputs, modals, cards)
│   ├── context/        # React context providers for global state management
│   ├── hooks/          # Custom React hooks for business logic and data fetching
│   ├── i18n/           # Internationalization setup and translation files
│   ├── layout/         # Page wrappers, headers, footers, and sidebars
│   ├── lib/            # Utility functions, third-party library configurations, and helpers
│   ├── router/         # Application routing configuration and route guards
│   └── screens/        # Main views and page-level components
├── public/             # Static public assets served directly by the server
├── package.json        # Project metadata, scripts, and dependencies
└── README.md           # Project documentation
